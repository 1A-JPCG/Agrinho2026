# 🌱 Quiz Agroforte Sustentável para o Agrinho 2026

Um jogo de perguntas e respostas dinâmico focado em conscientizar e testar conhecimentos sobre práticas modernas do agronegócio sustentável. O projeto aborda temas como o Código Florestal, bioinsumos, tecnologia no campo e preservação ambiental.

## 🌟 Principais Recursos
* **Sistema Anti-Decoreba:** As alternativas são embaralhadas a cada rodada automaticamente.
* **Mecânica de Combo (Streak):** Acertos consecutivos multiplicam a pontuação do jogador.
* **Contra o Tempo:** Limite de 15 segundos por pergunta com revelação automática da resposta.
* **Design Moderno:** Interface responsiva, fluida e com animações de transição suaves.

## 🛠️ Tecnologias
Construído de forma 100% nativa com **HTML5**, **CSS3** (Variáveis e Animações) e **JavaScript (ES6+)**.
